# Conner – wizytówka GitHub Pages

Gotowe statyczne repozytorium wizytówki kontaktowej.

## Pliki
- `index.html` – główna strona wizytówki
- `logo-white.svg` – logo Conner
- `antoni-kuran.vcf` – kontakt do pobrania na telefon

## Publikacja na GitHub Pages
1. Utwórz nowe repozytorium na GitHubie, np. `conner-wizytowka`.
2. Wrzuć wszystkie pliki z tej paczki do głównego katalogu repozytorium.
3. Wejdź w `Settings` → `Pages`.
4. Wybierz `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Zapisz. Po chwili GitHub pokaże adres strony.

## QR
Po uzyskaniu adresu GitHub Pages wygeneruj kod QR prowadzący do tej strony i pokaż go na telefonie lub wydrukuj na fizycznej wizytówce.
